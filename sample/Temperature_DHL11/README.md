## Measure Temperature and Humidity


## Setup
- Install 'DHL' Library
```
https://github.com/adafruit/DHT-sensor-library
```

- Install 'Adafruit_Sensor'
```
https://github.com/adafruit/Adafruit_Sensor
```

> Move Libraries to Arduino IDE directory => ./libraries/
